ABOUT

About Electryone

Electryone is a programming project developed by four high school students. The main idea behind the game was creating a battle royale which is set in space.

We wanted to create a game in which you can use newtonian physics to your advantage. Unfortunately, using a game engine was not an option, thus we had to build the whole system from scratch.

This page displays a lot of behind the scenes content about the game.

3D MODELS

Guns

The guns in Electryone are closely tied to the rules of the EWES system. There are four kinds: smg, shotgun, rifle and a sniper. All modelling was done in Blender, an open source 3D software.

Characters

Electryone currently has a female and a male character. The models were sculpted and textured in Blender, rigging was done with Adobe's free 3D rigger software called Mixamo. The animations we're using are also from the Mixamo library. Of course, if we had more time we would have created the animations by ourselves, but due to the lack of time we had to turn to such measures. Fortunately, Mixamo's services are free to use under creative commons.
